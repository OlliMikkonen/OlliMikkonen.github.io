Cash Tracker

Kouluprojekti

Cash Tracker

Projektin aikana oppia kertyi omista vastuistani, jotka olivat sovelluksen frontendin koodaaminen sekä testaus. Tämän lisäksi projektissa työskentely opetti todella paljon useamman henkilön tiimissä työskentelystä, suunnittelusta, riskien tunnistamisesta, versionhallinnasta, projektinhallinnasta sekä yleisesti kaikista sovelluskehityksen työvaiheista.

Käytettyjä teknologioita ja menetelmiä:

  • HTML
  • CSS
  • JavaScript
  • React Native
  • Scrum
  • Git
  • PHP

Frontend

React Native

Projektin frontend päätettiin toteuttaa käyttäen React Nativea sen suosion ja tiimin jäsenten kiinnostuksen vuoksi. Olin vastuussa sovelluksen frontendistä yhdessä tiimin toisen jäsenen kanssa. Alussa työskentelimme parikoodauksen periaatteella yhdessä ja tämän jälkeen siirryimme työstämään sovelluksen eri osioita.

Käytettyjä teknologioita ja menetelmiä:

  • React Native
  • Android Studio
  • JSX
  • Git
  • CSS

Projektinhallinta ja versionhallinta

GitHub, ZenHub, Scrum

Projektia lähdettiin tekemään Scrumin mallin mukaan kahden viikon sprinteissä. Projektinhallintaan käytettiin ZenHub työkalua ja versionhallinta suoritettiin Gitin avulla käyttäen GitHubia.

Käytettyjä teknologioita ja menetelmiä:

  • Scrum
  • Git
  • GitHub
  • ZenHub

Testaus

Järjestelmätestaus, konseptitestaus

Kaikki projektin testaaminen oli minun vastuullani. Suunnittelin miten projektissa suoritetaan testausta ja suoritin testauksen suunnitelmani mukaan. Sprintin tavoitteet testattiin minun toimestani jokaisen sprintin päätteeksi, raportoitiin ja havaitut bugit listattiin bugiraporttiin. Tämän lisäksi suunnittelin ja suoritin myös konseptitestausta.

Käytettyjä teknologioita ja menetelmiä:

  • Excel
  • Word
